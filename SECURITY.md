# Security Policy

## Supported Versions

This repository is in active development. The latest default branch is the supported version.

## Reporting a Vulnerability

Please do not open public issues for security-sensitive problems.

Report vulnerabilities privately to the repository owner through GitHub security reporting or a private contact method controlled by `D7Pr`.

When reporting, include:

- a short description of the issue
- impact and affected area
- reproduction steps if available
- any suggested mitigation

I will aim to acknowledge valid reports quickly and coordinate a fix before public disclosure.
